# Changelog

All notable changes to Connect4_LS will be documented in this file.

## [0.6.0] - 2026-07-13

### First beta release

- Added standard 6 × 7 Connect Four gameplay.
- Added Human vs Human, Human vs AI, and AI vs AI match modes.
- Added depth-selectable Numba-accelerated Lookahead players.
- Added the PPO_2004 neural-network player.
- Added LA13 move hints with evaluation scores.
- Added move analysis and AI status information.
- Added AI-vs-AI pause, resume, and single-step controls.
- Added falling-disc animation and sound effects.
- Added configurable display, gameplay, analysis, and audio settings.
- Added persistent per-user settings.
- Added session score tracking and configurable starting-player rules.
- Added a responsive title screen with artwork.
- Added a scrollable About screen with external links.
- Added a portable Windows 64-bit PyInstaller distribution.

This is the first public beta. The discs appear to fall in the correct
direction, which is already more than some early neural-network experiments
managed consistently.
